{:same; ... :}
====================================

Output code if same.

```html
{:same; /literal/literal/ >> code :}

{:same; /{:;varname:}/42/ >> ... :}
```
Any delimiter can be used:

```html
{:replace; ~a~b~ >> ... :}
{:replace; #a#b# >> ... :}
{:replace; |a|b| >> ... :}
{:replace; XaXbX >> ... :}
...
```
